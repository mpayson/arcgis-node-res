### Resources and Playground for ArcGIS & Node

To get started:
1. `$ git clone https://github.com/mpayson/arcgis-node-res.git`

For each sample:
1. `$ cd <sample>`
2. `$ npm install`
3. `$ npm start`

Samples:
- `stream-service`: playground for streaming locations with websockets and displaying with ArcGIS JS API
